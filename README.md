# projekt_ml

(data su rozdelene prvych 70% train, dalsich 15% val a poslednych 15% test)

Subor PL_strategy_V2 vypocita PL pri zvolenej strategii a Sharpov Pomer

Subor NN_model_v2 vytovori pozadovane modely
